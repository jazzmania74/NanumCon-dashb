나눔경영컨설팅 마케팅 인사이트 대시보드
=======================

GA4 데이터를 기반으로 웹사이트 트래픽을 분석하고 마케팅 인사이트 및 액션 플랜을 자동 생성하는 대시보드입니다.

주요 기능:
- KPI 7개 지표 (세션수, 활성사용자, 참여율, 새사용자 등) 및 전기 대비 증감률
- Chart.js 차트 6종 (일별 트렌드, 디바이스, 이벤트, 국가, 소스, 도시)
- 랜딩 페이지 성과 테이블 (이탈률 시각화)
- 페이지별 조회수 테이블
- 데이터 기반 마케팅 인사이트 자동 생성 (3~6개)
- 데이터 기반 액션 플랜 자동 생성 (4개)
- Looker Studio 원본 대시보드 임베딩
- Google Apps Script + GA4 API 연동 (JSONP)
- 새로고침 버튼으로 실시간 데이터 갱신

기술:
- 프론트엔드: HTML/CSS/JS (단일 파일)
- 차트: Chart.js 4.4.7
- 백엔드: Google Apps Script (GA4 Analytics Data API)
- 데이터 통신: JSONP callback
- 배포: GitHub Pages

제작: 김종혁 (나눔경영컨설팅)
